Core Operations
1. Database Queries (CRUD)
Use the table() method to target specific tables, chaining methods for filtering and execution. 

Select: supabase.table("countries").select("*").execute()
Insert: supabase.table("countries").insert({"name": "Denmark"}).execute()
Update: supabase.table("countries").update({"name": "Australia"}).eq("id", 1).execute()
Delete: supabase.table("countries").delete().eq("id", 1).execute()
2. Authentication
Manage user sessions directly through the auth property. 

Sign Up: supabase.auth.sign_up({"email": "user@example.com", "password": "secret"})
Sign In: supabase.auth.sign_in_with_password({"email": "user@example.com", "password": "secret"})
Sign Out: supabase.auth.sign_out()
3. Storage
Upload, download, and manage files in Supabase Storage buckets. 

Upload: supabase.storage.from_("avatars").upload("public/avatar.png", file_object)
Download: supabase.storage.from_("avatars").download("public/avatar.png")
Public URL: supabase.storage.from_("avatars").get_public_url("public/avatar.png")
4. Realtime
Subscribe to database changes using the RealtimeClient or the on() method on tables. 

def on_change(payload):
    print("Change received!", payload)

supabase.table("countries").on("*", on_change).subscribe()

5. Edge Functions
Invoke Supabase Edge Functions (Deno) using the functions property. 

response = supabase.functions.invoke("hello-world", invoke_options={"body": {"name": "Functions"}})

Asynchronous Support
For async applications (e.g., FastAPI), use the create_async_client method. 

import asyncio
from supabase import create_async_client

async def main():
    supabase = await create_async_client(url, key)
    response = await supabase.table("countries").select("*").execute()
    print(response.data)

asyncio.run(main())

Type Safety
You can generate Python type definitions from your database schema using the Supabase CLI:

npx supabase gen types --lang=python --project-id "$PROJECT_REF" --schema public > database.types.py

This generates classes like PublicCountries for parsing results and PublicCountriesInsert for formatting insert arguments, ensuring type safety in your Python code.
